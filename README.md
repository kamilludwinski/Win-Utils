# WinUtil

Windows utilities to make lives easier

## Features

- Manage apps (kill), includes PID and App Name search
- Manage open ports (kill the app), includes Port and App Name search.
```Useful if there is a hanging port that was left open by an app```

## Requirements: 
- **Windows 11 (x64).

## Dev 

### Requirements
- **.NET 8 SDK**

### Running app
```bash
./dev.sh
```

### Running tests
```bash
./dev.sh test
```
